Capture The Flag!

Level 0

The instructions for the next level are, of course, top secret. Hence, they can only be made available in encrypted form. Likewise, we want to ensure that only authorized parties have access to these instructions.

We accomplish this by using Pretty Good Privacy (PGP). If you don't know how to use PGP yet, maybe this link will help.

Your team needs to send an email encrypted with this key to jschauma@stevens.edu with a subject of "[CS615] level 0". To this email, attach a tar archive containing:

  • a README file containing the name of your team as well as key IDs for all members
  • the public PGP keys for all team members, named "$user-key.asc" (where "$user" is the Stevens username of the user)
  • detached signatures of the README as per the general instructions

Upon successful completion of this task, all team members will receive encrypted instructions on how to proceed to level 1.
