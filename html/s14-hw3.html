CS615A -- Aspects of System Administration - Spring 2014 - HW#3

HW#3: ifconfig output parsing

Summary:

Every System Administrator sooner or later has to write a program that needs to extract information from the output of the ifconfig(8) command. Your assignment is to determine the correct commands to extract the following information from the output of the ifconfig(8) command on the various platforms we use.

In particular, you should submit a shell script that will reliably extract:

  • all IP addresses
  • all MAC addresses
  • all netmasks

This assignment is worth 25 points.

Details:

Your script will behave exactly as outlined in the manual page ifcfg-data(1).

Example input and output:

	options=3<RXCSUM,TXCSUM>

The tool you write will need to work correctly on the following platforms:

Programming language

This assignment requires you to write a bourne shell script (!= bash(1)). As such, you can use any of the common commands found on the systems in questions, but it should not be written in a higher level language.

Group work

This assignment will be done in groups of three or four students. When working as a group, each group member will be responsible for one of the platforms noted above. When you submit your work, please note in the README file which student worked on which platform. (The two BSDs and the two Linux flavors can be grouped together.)

Deliverables and Due Date

You will submit a single tar(1) archive. The file to submit will be called "$USER-hw3.tar" (where "$USER" is your username). The archive will extract all files and subdirectories into a directory named $USER. Only one group member needs to submit the file.

You will only submit a single shell source file as well as a simple text file called README in which you may provide commentary on your program. Commentary elaborating on the scalability or limitations of the tool in general and your implementation in particular is welcome.

The folllowing is an example sequence of commands to create a valid homework submission: 

mkdir $USER
cd $USER
vi README
vi ifcfg-data
[code code code]
cd ..
tar cf $USER-hw3.tar $USER
